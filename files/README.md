# 🏛️ قاعدة المعرفة الضريبية العقارية

أداة بحث ذكية لخدمة العملاء — مربوطة بـ Google Sheets مباشرة.

---

## 🗂️ خطوة 1 — تجهيز Google Sheets

### إنشاء الشيت

افتح Google Sheets وأنشئ شيت جديد. الصف الأول هو **أسماء الأعمدة** بالضبط كالتالي:

| العمود | الاسم | الوصف |
|--------|-------|-------|
| A | `id` | رقم فريد لكل سطر (1، 2، 3...) |
| B | `category` | التصنيف (مثال: السداد، الإعفاءات، الإقرارات) |
| C | `question` | السؤال كما سيظهر في البحث |
| D | `legal` | النص القانوني الرسمي بالكامل |
| E | `simple` | الرد المبسط للعميل بالعامية المصرية (ممكن تسيبه فاضي والموقع يولده تلقائياً) |
| F | `steps` | الخطوات مفصولة بـ `\|` (اختياري) |
| G | `source` | المصدر القانوني (مثال: قانون رقم 196 لسنة 2008) |

> **ملاحظة:** الأعمدة تدعم العربية والإنجليزية. استخدم أي أسماء من الجدول أعلاه.

---

### مشاركة الشيت

1. افتح الشيت
2. اضغط **Share** (مشاركة)
3. غيّر من **Restricted** إلى **Anyone with the link**
4. تأكد إن الصلاحية **Viewer**
5. انسخ الـ **Sheet ID** من الرابط:
   ```
   docs.google.com/spreadsheets/d/ ➜ THIS_IS_YOUR_SHEET_ID ➜ /edit
   ```

---

## ⚙️ خطوة 2 — الحصول على Gemini API Key

1. روح [Google AI Studio](https://aistudio.google.com/app/apikey)
2. اضغط **Create API Key**
3. انسخ المفتاح

> **ملاحظة:** Gemini API مجاني بحدود سخية (60 طلب/دقيقة في الـ free tier).

---

## 🚀 خطوة 3 — نشر على Vercel

### ربط GitHub بـ Vercel

1. ارفع الكود على GitHub
2. روح [vercel.com](https://vercel.com) → **New Project**
3. اختار الـ repository
4. **قبل النشر:** روح **Environment Variables** وأضف:

| الاسم | القيمة |
|-------|--------|
| `GOOGLE_SHEET_ID` | الـ ID من رابط الشيت |
| `GEMINI_API_KEY` | مفتاحك من Google AI Studio |

5. اضغط **Deploy** ✅

## 🧠 كيف تشتغل البحث؟

### Pipeline الجديد (بدون Embeddings):

```
سؤال المستخدم
       ↓
   تطبيع عربي
(أ إ آ → ا، ة → ه، إزالة تشكيل)
       ↓
  تحويل العامية
(عايز→أريد، امتى→موعد، ...)
       ↓
  ترتيب بالكلمات المفتاحية
(السؤال أعلى وزن، ثم التصنيف، ثم القانون)
       ↓
  أفضل 10 مرشحين فقط
       ↓
  Gemini يختار الأقرب
       ↓
     النتيجة
```

| الوضع | متى بيحصل |
|-------|-----------|
| **بحث ذكي (Gemini)** | لما `GEMINI_API_KEY` مضبوط |
| **بحث احتياطي** | لما الـ API مش متاحة — بحث بالكلمات |

### مميزات Pipeline الجديد:
- ✅ لا يرسل قاعدة البيانات كاملة لـ Gemini (أسرع وأوفر)
- ✅ يفهم العامية المصرية (قاموس 100+ كلمة)
- ✅ يطبّع الحروف (أ إ آ، ة، ى)
- ✅ محمي ضد Prompt Injection
- ✅ لا يخترع معلومات قانونية
